1. Building a website with HTML

Project description

  • Create a webiste using HTML.
  • Upload the website to Github using Git bash.
  • Document the process.

Creating the website

As a total HTML beginner, I began by watching several YouTube tutorials for beginners (links below) and the videos made by Hafliði, the teacher. I also read about HTML basics in W3schools. The website was edited in Brackets, a web development software. The template for the design was downloaded from HTML5UP and is called Hyperspace.

Before I started editing in Brackets, I made a sketch of the website layout.

I made some changes to the Hyperspace template, for example the clickable buttons in the portfolio.

Uploading the website

I downloaded Git , an open and free source distributed version control system. I then signed up on GitHub. I created a new repository and made sure it was public.


I used Git Bash to commit the website to the new repository in GitHub.
